# Инструкция по установке VolleyScore Master

## Системные требования

### Windows

- **Операционная система:** Windows 10 или новее (64-bit)
- **Свободное место:** минимум 100 МБ на диске
- **Сеть:** для интеграции с vMix требуется доступ к компьютеру с vMix по локальной сети

## Установка для Windows

### Шаг 1: Скачивание установщика

1. Перейдите на страницу [Releases](https://github.com/ivmoiseev/vmix-volley-scoreboard/releases) в репозитории проекта
2. Найдите последнюю версию приложения (например, `v2.0.0`)
3. Скачайте файл `VolleyScore Master Setup X.X.X.exe` (где X.X.X — номер версии)

### Шаг 2: Запуск установщика

1. Найдите скачанный файл в папке "Загрузки" (или в том месте, куда вы его сохранили)
2. Дважды щелкните по файлу `VolleyScore Master Setup X.X.X.exe`
3. Если появится предупреждение Windows о безопасности:
   - Нажмите "Подробнее"
   - Нажмите "Выполнить в любом случае"
   - Это нормально, так как приложение не подписано цифровой подписью

### Шаг 3: Установка

1. Откроется мастер установки
2. Выберите язык установки (если доступно)
3. Прочитайте лицензионное соглашение и примите его
4. Выберите папку для установки:
   - По умолчанию: `C:\Program Files\VolleyScore Master`
   - Вы можете выбрать другую папку, нажав "Обзор"
5. Выберите дополнительные опции:
   - ☑ Создать ярлык на рабочем столе (рекомендуется)
   - ☑ Добавить в меню "Пуск" (рекомендуется)
6. Нажмите "Установить"
7. Дождитесь завершения установки
8. Нажмите "Готово" для завершения

### Шаг 4: Первый запуск

1. После установки приложение может запуститься автоматически
2. Если нет, запустите его одним из способов:
   - Дважды щелкните по ярлыку на рабочем столе
   - Найдите "VolleyScore Master" в меню "Пуск" и запустите
3. При первом запуске может появиться предупреждение Windows Defender или другого антивируса:
   - Это нормально для неподписанных приложений
   - Нажмите "Разрешить" или "Разрешить доступ", если появится запрос

## Проверка установки

После установки убедитесь, что:

1. ✅ Приложение запускается без ошибок
2. ✅ Открывается главное окно с кнопками "Создать новый матч" и "Открыть существующий матч"
3. ✅ В меню "Пуск" есть ярлык приложения (если вы выбрали эту опцию)
4. ✅ На рабочем столе есть ярлык (если вы выбрали эту опцию)

## Обновление приложения

### Автоматическое обновление

Приложение поддерживает **автоматические обновления** через GitHub Releases:

- При запуске приложение проверяет наличие новой версии (с задержкой несколько секунд)
- Если обновление найдено, можно загрузить и установить его через интерфейс (страница «О программе» или уведомление)
- Автоматическую проверку обновлений можно отключить в настройках на странице «О программе»
- Подробности настройки для разработчиков: см. [docs/development/auto-updates-setup.md](docs/development/auto-updates-setup.md)

### Ручное обновление

Если автоматическое обновление отключено или не сработало:

1. Скачайте новую версию установщика с [GitHub Releases](https://github.com/ivmoiseev/vmix-volley-scoreboard/releases)
2. Запустите новый установщик
3. Установщик автоматически удалит старую версию и установит новую
4. Ваши настройки и сохранённые матчи сохранятся

При необходимости можно удалить старую версию через «Панель управления» → «Программы и компоненты», затем установить новую по инструкции выше.

## Удаление приложения

### Через Панель управления

1. Откройте "Панель управления" → "Программы и компоненты" (или "Параметры" → "Приложения")
2. Найдите "VolleyScore Master" в списке
3. Нажмите "Удалить" (или "Удалить" → "Удалить")
4. Следуйте инструкциям мастера удаления

### Что удаляется

- ✅ Файлы приложения
- ✅ Ярлыки на рабочем столе и в меню "Пуск"

### Что сохраняется

- ✅ Сохраненные матчи (файлы в папке `matches/`)
- ✅ Логотипы команд (файлы в папке `logos/`)
- ✅ Настройки приложения (файл `settings.json`)

Эти данные находятся в папке пользователя и не удаляются при удалении приложения.

## Решение проблем при установке

### Проблема: "Windows защитил ваш компьютер"

**Решение:**
1. Нажмите "Подробнее"
2. Нажмите "Выполнить в любом случае"
3. Это нормально для неподписанных приложений

### Проблема: "Не удалось установить приложение"

**Возможные причины и решения:**

1. **Недостаточно прав администратора:**
   - Щелкните правой кнопкой по установщику
   - Выберите "Запуск от имени администратора"

2. **Антивирус блокирует установку:**
   - Временно отключите антивирус
   - Добавьте папку установки в исключения антивируса
   - После установки включите антивирус обратно

3. **Недостаточно места на диске:**
   - Освободите минимум 100 МБ на диске
   - Выберите другой диск для установки

### Проблема: "Приложение не запускается после установки"

**Возможные причины и решения:**

1. **Отсутствуют необходимые библиотеки:**
   - Убедитесь, что у вас установлена последняя версия Windows 10/11
   - Установите обновления Windows через "Центр обновления Windows"

2. **Конфликт с другим программным обеспечением:**
   - Попробуйте запустить приложение в режиме совместимости
   - Проверьте, не блокирует ли антивирус запуск

3. **Ошибка при запуске:**
   - Проверьте логи ошибок (если доступны)
   - Переустановите приложение
   - Создайте Issue на GitHub с описанием проблемы

### Проблема: "Не могу найти сохраненные матчи после переустановки"

**Решение:**
Сохраненные матчи находятся в папке пользователя и не удаляются при удалении приложения. Обычно они находятся в:
- `C:\Users\[ВашеИмя]\AppData\Roaming\VolleyScore Master\matches\`

Или в папке установки приложения (в зависимости от версии).

## Дополнительная информация

### Расположение файлов

После установки файлы приложения находятся в:

- **Исполняемые файлы:** `C:\Program Files\VolleyScore Master\` (или выбранная вами папка)
- **Пользовательские данные:** `C:\Users\[ВашеИмя]\AppData\Roaming\VolleyScore Master\`
  - `matches/` — сохраненные матчи
  - `logos/` — логотипы команд
  - `settings.json` — настройки приложения

### Требования для интеграции с vMix

Для работы с vMix необходимо:

1. Компьютер с установленным vMix должен быть доступен по сети
2. В vMix должен быть включен HTTP API (по умолчанию включен)
3. Компьютер с VolleyScore Master и компьютер с vMix должны быть в одной локальной сети
4. Необходимо знать IP адрес компьютера с vMix и порт HTTP API (по умолчанию 8088)
