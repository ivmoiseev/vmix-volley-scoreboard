# Руководство по тестированию

## Функциональное тестирование

### 1. Управление матчами

#### Создание матча
- [ ] Создать новый матч через меню (Ctrl+N)
- [ ] Создать новый матч через кнопку на главной странице
- [ ] Проверить, что создается корректная структура данных
- [ ] Проверить генерацию уникального ID матча

#### Открытие матча
- [ ] Открыть существующий матч через меню (Ctrl+O)
- [ ] Открыть матч через кнопку на главной странице
- [ ] Проверить обработку несуществующего файла
- [ ] Проверить обработку поврежденного JSON файла
- [ ] Проверить валидацию данных при открытии

#### Сохранение матча
- [ ] Сохранить матч через меню (Ctrl+S)
- [ ] Сохранить матч через кнопку на странице управления
- [ ] Сохранить матч как... (Ctrl+Shift+S)
- [ ] Проверить обработку ошибок при сохранении
- [ ] Проверить обновление времени изменения

### 2. Ведение счета

#### Управление счетом
- [ ] Увеличить счет команды А (+1)
- [ ] Увеличить счет команды Б (+1)
- [ ] Уменьшить счет команды А (-1)
- [ ] Уменьшить счет команды Б (-1)
- [ ] Проверить, что счет не может быть отрицательным
- [ ] Проверить автоматический переход подачи при начислении очка

#### Управление подачей
- [ ] Проверить автоматический переход подачи
- [ ] Изменить подачу вручную (кнопки ◄ ►)
- [ ] Проверить отображение текущей подачи

#### Сетбол и матчбол
- [ ] Проверить определение сетбола (24:24 и выше)
- [ ] Проверить определение матчбола (5-я партия, счет 2:2, сетбол)
- [ ] Проверить отображение индикаторов

#### Завершение партии
- [ ] Завершить партию при счете 25:23
- [ ] Попытаться завершить партию при счете 24:23 (должна быть ошибка)
- [ ] Проверить переход к следующей партии
- [ ] Проверить определение победителя партии для подачи в следующей партии
- [ ] Проверить завершение матча (3 выигранные партии)

#### Отмена действий
- [ ] Отменить последнее действие
- [ ] Проверить восстановление предыдущего состояния
- [ ] Проверить, что история очищается после завершения партии

### 3. Расширенная статистика

#### Включение/выключение
- [ ] Включить расширенную статистику
- [ ] Выключить расширенную статистику
- [ ] Проверить, что статистика не блокирует основное ведение счета

#### Управление статистикой
- [ ] Добавить атаку команде А
- [ ] Добавить блок команде Б
- [ ] Добавить подачи
- [ ] Добавить ошибки соперника
- [ ] Проверить, что значения не могут быть отрицательными
